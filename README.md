# Analyse de la performance et de l’accessibilité des services publics présenté par TSENGLE Kokou Joassin

## Contexte
Ce projet a été réalisé dans le cadre d’un test de recrutement Data Analyst.  
Il vise à analyser la demande de services publics administratifs, la capacité des centres et leur répartition territoriale à partir de données opérationnelles.

## Objectifs
- Évaluer la pression de la demande par territoire
- Mesurer la performance opérationnelle des centres
- Analyser l’accessibilité aux services publics
- Produire des indicateurs décisionnels clairs

## Démarche
- Analyse exploratoire des données (EDA)
- Nettoyage et préparation des données
- Construction de KPI
- Visualisation et modélisation sous Power BI

## Structure du projet
- `data/raw` : données brutes
- `data/processed` : données nettoyées
- `eda` : analyse exploratoire
- `cleaning` : rapport de nettoyage
- `kpi` : définition des indicateurs
- `dashboard` : fichier Power BI
- `presentation` : présentation décideurs
- `report` : rapport de synthèse

## Livrables
- Dashboard Power BI interactif
- Présentation PowerPoint (5 slides)
- Rapports d’analyse et de synthèse

## Outils
Power BI, Excel, Git
